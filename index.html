Project section

Project Name

Online metronome

This is a metronome that can be used by anyone who has an internet connection.

What is this project about?

A metronome is an incredibly useful tool when practicing any kind of music, and having access to a metronome anywhere you have an internet connection is extremely helpful.

The skills used in creating this project are:

  1. using css and html to create a blinking light that synchs to a timecode
  2. using javascript to play a beep that synchs to the same timecode
  3. using an html and css interface to adjust the rate at which the metronome beeps

Who is this project for and why did I choose this project?

This project is for people who need quick acess to a metronome and do not want to install any apps or programs to their phone or computer.


I chose to create this project becuase I have found myself needing a metronome and do not want to download an app to use it. It is also very helpful in teaching situations where you need a metronome on the fly.
